# Further Exploration

Zim's manual is well written and has a lot of useful advice in it. Pay particular attention to the [Usage](http://zim-wiki.org/manual/Usage.html) and [Plugins](http://zim-wiki.org/manual/Plugins.html) sections. If you're inclined to write scripts that *create* Zim notebooks or pages, be sure to check out the [Wiki Syntax](http://zim-wiki.org/manual/Help/Wiki_Syntax.html) section.

Zim also has a publicly shared [wiki with Zim tips and tricks](https://github.com/jaap-karssenberg/zim-wiki/wiki/How-to-upload-an-image-to-this-wiki) hosted on GitHub. If you have anything to contribute, don't hesitate to registar a GitHub account and edit this wiki.

You can get help with Zim by signing up for the [Zim mailing list](https://launchpad.net/~zim-wiki) hosted at Launchpad and also in the chat room `#zim-wiki` on [Freenode](https://www.freenode.net/) IRC network. I'm the host in that chat room and if I'm at my desk I will respond. (Be prepared to wait a while for a response.)

If you find a bug and you're sure it's a bug, please report it in the [bug tracker](http://bugs.launchpad.net/zim) on Launchpad. If you're not sure it's a bug, post on the mailing list asking for help first.
